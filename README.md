## 简介
本程序是基于MFC的串口通信程序，仿SSCOM界面编写
含以下功能：
1. 端口号、波特率、数据位、停止位、校验位设置
2. 与串口进行通信
3. 数据Hex发送 与 接收数据Hex显示
4. 打开文件（文件内容预览）、发送文件、停止发送文件
